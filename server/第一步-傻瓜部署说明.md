# 安全大师 · 服务端「第一步」傻瓜操作说明

> 读完这一段你只需要知道：**做完后，用浏览器能打开一个测试页，就说明「服务端程序」已经在你的腾讯云里跑起来了。**

---

## 这一步做完你会得到什么？

- 你的云服务器上会跑一个很小的程序。
- 用浏览器访问 **「服务器 IP + 端口」** 时，能看到一行 **「运行中」** 的提示（**不需要备案也能测**，因为用的是 IP，不是域名）。

---

## 你需要准备什么？

- 能登录 **宝塔面板**（网页）。
- 会 **复制、粘贴** 文字。
- 本机（你的 Mac）里已经有这个文件夹里的两个文件：`package.json`、`index.js`（在项目 `server` 目录里）。

---

## 操作步骤（请按顺序做）

### 第 1 步：在宝塔里安装 Node.js

1. 登录宝塔 → 左侧点 **「软件商店」**。
2. 搜索 **「Node」** 或 **「Node 版本管理器」**。
3. 安装 **Node 版本管理器**（或宝塔提供的 **Node.js 一键安装**），版本选 **20 或 22 这类 LTS（长期支持版）**。
4. 等安装完成（可能要几分钟）。

**成功标志**：软件商店里能看到 Node，或左侧出现 **「Node 项目」** 之类菜单（不同版本宝塔界面可能略有差别）。

---

### 第 2 步：在服务器上建一个文件夹

1. 宝塔左侧点 **「文件」**。
2. 进入 **`/www/wwwroot`**（没有就自己点路径进去）。
3. 点 **「新建文件夹」**，名字填：**`safemaster-api`**（全小写，中间是英文减号也可以，只要你自己记住）。
4. 进入这个 **`safemaster-api`** 文件夹，保持空白即可。

---

### 第 3 步：把两个文件上传到服务器

1. 在你自己的电脑上，找到项目里的 **`server`** 文件夹。
2. 把里面的 **`package.json`** 和 **`index.js`** 两个文件，用宝塔 **「上传」** 按钮上传到 **`/www/wwwroot/safemaster-api`**。
3. 上传完后，这个文件夹里应该正好有这两个文件。

**成功标志**：在宝塔文件管理里点开能看到 `package.json` 和 `index.js`。

---

### 第 4 步：在宝塔里打开「终端」

1. 宝塔左侧点 **「终端」**（或 **「SSH 终端」**）。
2. 出现黑色/白色命令行窗口。

---

### 第 5 步：复制下面三段命令，**一段一段粘贴**，每段粘贴后按 **回车**，等它跑完再粘贴下一段。

**第一段（进入文件夹）：**

```bash
cd /www/wwwroot/safemaster-api
```

**第二段（安装依赖，可能要 1～2 分钟）：**

```bash
npm install
```

**第三段（启动程序）：**

```bash
node index.js
```

**如果第三段运行后，最后一行出现类似：**

`SafeMaster API 监听 http://0.0.0.0:3000`

说明程序已经在你这台服务器上跑起来了。

> **注意**：这个窗口**不要关**，关了程序就停了。以后我们会用 **PM2** 让它在后台一直跑（下一步再教你）。

---

### 第 6 步：在腾讯云放行端口 3000

1. 打开 **腾讯云控制台** → **轻量应用服务器** → 点你的那台机器。
2. 找到 **「防火墙」**。
3. **添加规则**：端口填 **`3000`**，协议 **TCP**，备注随便写「API 测试」。
4. 保存。

---

### 第 7 步：用手机或电脑浏览器测试

1. 记下你的服务器 **公网 IP**（轻量控制台首页能看到，例如 `111.229.xxx.xxx`）。
2. 打开浏览器，地址栏输入（把 IP 换成你自己的）：

```text
http://你的公网IP:3000/health
```

3. 若看到一段 **JSON** 文字，里面有 `"ok":true` 和 `"SafeMaster API 运行中"`，**第一步就成功了**。

也可以试：

```text
http://你的公网IP:3000/
```

应能看到简单网页标题 **「SafeMaster 服务端已启动」**。

---

## 常见问题

**问：粘贴第三段后，提示 `command not found: node`？**  
→ 说明 Node 没装好或没在 PATH 里。回到 **软件商店** 确认 Node 已安装成功；或在宝塔 **Node 版本管理器** 里启用一个版本。

**问：浏览器打不开，一直转圈？**  
→ 检查 **防火墙** 是否已放行 **3000**；检查 **第三段命令是否还在运行**（终端没关）。

**问：域名 safemaster.top 还是打不开？**  
→ **备案没下来前，域名可能被拦截**，先用 **IP:3000** 测即可，这不影响我们先把程序跑通。
